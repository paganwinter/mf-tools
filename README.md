# MF Tools

Utilities for Indian Mutual Funds


## Screen
- https://paganwinter.github.io/mf-tools/screen-v2
- https://paganwinter.github.io/mf-tools/screen

old version: https://paganwinter.github.io/mf-tools/index.html

### Features
- Produces Average Rolling Returns for 1,2,3,..10 year windows for selected funds.
- Allows filtering funds by type or name.
- Allows adding funds to watchlist.

> If funds list is outdated, update [data/NAVOpen.txt](data/NAVOpen.txt) with https://www.amfiindia.com/spages/NAVOpen.txt
