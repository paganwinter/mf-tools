## MF Analysis Tools

Utility to analyse Indian Mutual Funds.
- Produces Average Rolling Returns for 1,2,3,..10 year windows for selected funds.
- Allows filtering funds by type or name.
- Allows adding funds to watchlist.

Work in progress
